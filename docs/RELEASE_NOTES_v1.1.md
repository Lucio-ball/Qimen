## 🎮 安装与使用

### 系统要求
- **操作系统：** Windows 10/11 (64位)
- **内存：** 4GB RAM (推荐 8GB)
- **存储空间：** 500MB 可用空间
- **显示器：** 1366×768 分辨率或更高

### 安装方法
1. **单文件版本**：直接运行 `奇门遁甲工作台.exe`
2. **完整版本**：解压 `奇门遁甲工作台` 文件夹，运行其中的 `.exe` 文件

### 首次使用
1. 双击运行应用程序
2. 使用 Ctrl+N 创建第一个案例
3. 通过 "设置" → "首选项" 自定义界面
4. 开始您的奇门遁甲学习之旅！
