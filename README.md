# 奇门遁甲工作台 v1.0 alpha

一款专为奇门遁甲专业人士和深度爱好者设计的、专业级、可高度定制的桌面分析平台。

![版本](https://img.shields.io/badge/版本-v1.0_alpha-blue)
![平台](https://img.shields.io/badge/平台-Windows-lightgrey)
![许可证](https://img.shields.io/badge/许可证-MIT-green)

## 🌟 核心功能

### 📁 多案例管理与文件式工作区
- **工作区概念**: 采用现代IDE风格的工作区管理，每个项目独立管理
- **案例持久化**: 支持.qmw格式的案例文件，数据完整保存
- **多案例并行**: 同时打开多个案例进行对比分析
- **智能文件浏览**: 内置案例浏览器，快速定位和管理案例

### 🎨 高度可定制的盘面显示系统
- **灵活布局**: 支持停靠面板的四周停靠和自由调整
- **主题配置**: 多种显示主题和配色方案
- **五行颜色**: 可配置的五行颜色显示系统
- **字体设置**: 值符/值使粗体等个性化显示选项

### 🏷️ 自动化用神标注模板
- **智能标注**: 基于特殊参数的自动用神识别
- **模板系统**: 可创建和保存自定义标注模板
- **图层管理**: 支持多图层标注，独立控制可见性
- **批量应用**: 一键应用预设模板，提高分析效率

### ⚙️ 专业级设置系统
- **分类配置**: 显示设置、通用设置、数据管理分类清晰
- **实时预览**: 设置修改即时生效，所见即所得
- **配置持久化**: 设置自动保存，重启后保持用户偏好

## 🚀 如何使用

### 方式一：直接运行（推荐）
下载并运行 `dist/奇门遁甲工作台.exe` 即可开始使用。

### 方式二：源码运行
```bash
# 克隆仓库
git clone https://github.com/Lucio-ball/Qimen.git
cd Qimen

# 安装依赖
pip install -r requirements.txt

# 运行程序
python run_gui.py
```

## 📋 系统要求

- **操作系统**: Windows 10/11 (64位)
- **内存**: 建议4GB以上
- **存储空间**: 至少100MB可用空间
- **显示器**: 分辨率1280x720以上

## 🎯 快速开始

1. **创建工作区**: 启动后选择"新建工作区"或"打开工作区"
2. **起局分析**: 点击"新建案例"，输入时间和事由信息
3. **标注分析**: 使用右侧标注面板添加分析标注
4. **保存案例**: 使用Ctrl+S保存当前分析结果
5. **模板应用**: 在"模板"菜单中选择预设模板快速标注

## 🎯 核心特性

### 马星冲动算法 V2
- 严格按照优先级顺序：本宫入墓 > 对宫入墓 > 本宫空亡 > 对宫空亡
- 支持"冲动所有"特性，同时处理多个目标
- 智能类型映射和视觉样式标记

### 精密样式控制
- 富文本显示支持
- 删除线样式自动应用
- 垂直/横向自适应布局
